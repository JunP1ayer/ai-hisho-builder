# AI秘書ビルダー

質問に答えるだけで、あなた専用の「AI秘書」（Claude Code環境）の初期設定一式を組み立てられるWebウィザードです。**Central AX の Claude Code 研修で配布している教材**です。

👉 **使う**: https://junp1ayer.github.io/ai-hisho-builder/

## 何ができるか

3つのステージを順に進めると、各ステージで「構築指示書＋あなたの決定」がまとまった `.md` ファイルをダウンロードできます。それを Claude Code に渡すだけで、あなた専用のAI秘書環境が組み上がります。

| ステージ | 決めること | できあがるもの |
|---|---|---|
| 01 ガワをつくる | 会社・名前・役割・連携ツール・権限 | CLAUDE.md／フォルダ構成／権限設定 |
| 02 能力をさずける | スキルとエージェントチーム | スラッシュコマンド（/daily-schedule 等） |
| 03 中身をそだてる | 読み込ませる会社資料 | context/（自社理解度スコア付き） |

## 使い方

1. 上のURLをブラウザで開く（インストール・API設定は不要です）
2. ステージ1から順に選択して「組み立てる」→ `.md` をダウンロード
3. Claude Code を開き、ダウンロードした `.md` ファイルを渡す
4. 3ステージ完了で、あなたのAI秘書が稼働します

入力内容はブラウザ内（localStorage）にのみ保存され、外部には送信されません。

## もっと本格的に始めたい方へ

```
claude plugin marketplace add JunP1ayer/claude-plugins
```

のあと `/first-setup` を実行すると、職種別の初期設定プラグインが使えます。
